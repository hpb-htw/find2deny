find2deny
*********


Tools to build Firewall Command for UFW from List of IPs


Installation
============

To install the latest release on `PyPI <https://pypi.org/project/find2deny/>`_,
simply run:

::

  pip install toml

Or to install the latest development version, run:

::

  git clone [TODO]
  cd find2deny
  python setup.py install
